## 🎓 Learning Resources

### CSS Responsive Design
- Mobile-first approach (start small, add up)
- Breakpoints: 480px, 600px, 768px, 992px
- Flexible layouts (flexbox, grid)
- Relative sizing (%, em, rem)

### Touch-Friendly Design
- 48px × 48px minimum buttons
- 12px+ gap between touch targets
- Avoid hover-only interactions
- Proper form field sizing

### Mobile Meta Tags
- viewport: Controls zoom and scaling
- apple-mobile-web-app-capable: iOS PWA
- theme-color: Address bar color
- viewport-fit: Safe area support
