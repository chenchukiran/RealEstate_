## 🚀 How to Test Locally

### Chrome DevTools (Easiest)
1. Open your website in Chrome
2. Press `F12` to open DevTools
3. Press `Ctrl+Shift+M` (Windows) or `Cmd+Shift+M` (Mac)
4. Select device from dropdown
5. Test all pages and interactions

### Real Device Testing
1. On same WiFi as your computer
2. Find your computer's IP address
3. Access: `http://YOUR_IP:PORT` on phone
4. Test all interactive elements
5. Check both portrait and landscape
